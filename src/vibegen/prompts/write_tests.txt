## OUTCOME
Your test file passes when ALL of the following hold:
1. Each item in the TEST PLAN has at least one corresponding test function.
2. For each tested function/method: one happy-path test, one edge-case test,
   one failure-mode test (raises expected exception or returns sentinel value).
3. All imports use the installed package name:
     Correct : from {{package}}.module import SomeClass
     WRONG   : from src.{{package}}.module import SomeClass
4. Only packages listed in INSTALLED DEPENDENCIES are imported (plus pytest/stdlib).
5. No test uses `assert True` or trivially-passing assertions.
