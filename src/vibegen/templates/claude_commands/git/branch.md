## Scope
Create or switch to a branch for: $ARGUMENTS

## Anchor
Run `git branch --show-current` and `git log --oneline -5` for current context.

## Outcome
Use naming convention: `feat/`, `fix/`, `chore/`, `docs/`, or `test/` prefix with kebab-case slug from $ARGUMENTS.
Run `git checkout -b <branch>` for new branches. Confirm with `git branch --show-current`.
